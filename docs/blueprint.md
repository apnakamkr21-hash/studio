# **App Name**: Campus Events Hub

## Core Features:

- Event Discovery: Browse a comprehensive list of campus events, workshops, and competitions with detailed information.
- Ticket Booking: Securely book and pay for event tickets using integrated payment gateways like Razorpay or Stripe.
- Digital Tickets: Receive digital tickets with unique QR codes for easy entry and attendance tracking.
- Event Management Dashboard: Allow event organizers to create events, set prices, manage bookings, track payments, and monitor attendance.
- Automated Recommendations: AI-powered recommendations for events based on student interests and past activities; acts as a tool to help users find suitable events.
- Committee Dashboards: Individualized dashboards for each campus committee to manage their events and participants.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to evoke a sense of community and campus spirit.
- Background color: Light Lavender (#E6E6FA) for a clean and calming backdrop.
- Accent color: Golden Yellow (#FFD700) to highlight important actions and information.
- Headline font: 'Space Grotesk' sans-serif for a techy and scientific look, well-suited for event titles.
- Body font: 'Inter' sans-serif for clean and modern readability in event descriptions.
- Use clear, modern icons to represent event categories and actions.
- Subtle animations and transitions to provide feedback and enhance user experience when browsing or booking events.
